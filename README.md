# Go notları

Bu repo Go için [Tour of go](https://go.dev/tour/welcome/1) içerisinde anlatılan konuların özetlenmiş bir halidir. Notlar türkçe olarak hazırlanmıştır. Go öğrenirken bir cheat-sheet olarak kullanılabilir. `main.go` içerisinde öğrenilen konuların uygulaması yapılabilir. Kodlarınızı yazdıktan sonra `go run .` komutu yeterli olacaktır.
